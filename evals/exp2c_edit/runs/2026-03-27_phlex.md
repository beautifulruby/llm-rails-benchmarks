## Why More Lines Than Vanilla?

The Phlex DSL requires more Ruby code to express the same HTML:

```ruby
# Phlex form
form(action: path, method: "post", class: "space-y-4") do
  input(type: "hidden", name: "_method", value: "patch")
  textarea(name: "comment[body]", rows: 3, class: "...") { @comment.body }
end

# vs ERB form
<%= form_with model: [post, comment], method: :patch do |f| %>
  <%= f.text_area :body, rows: 3, class: "..." %>
<% end %>
```

Rails form helpers are more concise than raw HTML-like DSL.
