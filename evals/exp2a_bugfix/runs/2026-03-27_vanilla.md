## Root Cause Identified

Turbo intercepts form submissions via AJAX. Delete was working (DB updated) but DOM wasn't updated because controller responded with redirect instead of Turbo Stream.

## Solution Implemented

```ruby
def destroy
  if params[:confirmation] == @comment.excerpt(30)
    comment_id = @comment.id
    @comment.destroy
    respond_to do |format|
      format.turbo_stream { render turbo_stream: turbo_stream.remove("comment-#{comment_id}") }
      format.html { redirect_to @post, notice: "Comment was successfully deleted." }
    end
  else
    redirect_to @post, alert: "Confirmation text did not match."
  end
end
```
